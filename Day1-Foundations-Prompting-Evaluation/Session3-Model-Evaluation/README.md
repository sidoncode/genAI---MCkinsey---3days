# Session 3: Model Evaluation and Comparison

**Time:** 1:30 – 3:15 (1 hour 45 minutes)

## Overview

This session covers how to systematically evaluate and compare LLM models for agentic consulting tasks. Participants will learn to define evaluation metrics, use LLM-as-a-judge patterns, benchmark response quality, analyze cost/latency trade-offs, and build A/B testing frameworks — all using **McKinsey consulting evaluation criteria** like strategic relevance, MECE structure, analytical rigor, and executive readiness.

## Key Topics

- Defining McKinsey-quality evaluation criteria for consulting AI
- LLM-as-a-judge automated evaluation of consulting deliverables
- Response quality benchmarking on consulting prompts
- Latency, cost, and throughput analysis
- A/B testing and comparison frameworks
- Scikit-learn classification metrics (precision, recall, F1) for client feedback
- DeepEval and G-Eval for LLM-scored evaluation
